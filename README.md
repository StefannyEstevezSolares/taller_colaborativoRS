# Tienda GitHub


La aplicación permite administrar los productos de una tienda.

## Tabla de contenido

|Número|Contenido|Descripción|
|---|---|---|
|1|Agregar producto|se agrega nombre, precio, cantidad|
|2|Listar producto|enlista los productos agregados|
|3|Actualizar cantidad|ingresa nombre y nueva cantidad

## Tecnología utilizada

>**Python**


## Uso

- La aplicación se ejecuta mediante un menú interactivo por consola. 

- El programa se mantine en ejecución hasta que el usuario sleccione la opción Salir.

## Instrucciones de ejecución

### Menu Principal
Será solo de presionar números o se requiere el input de alguna palabra principal. Se deben llenar datos
#### SubMenú ###
Como se utilizan los súbmenú de haber alguno.

#### Comando de ejecución del programa
``` 
main ejemplo.py
```

> [!NOTE] 
>Los menú son interactivos y requieren que siga las instrucciones


## Estructura del Proyecto
- El aplicativo esta desarrolado en Python
- El aplicativo se ejecuta desde consola
- Dentro del programa esta diseñado con funciones, condicionales, ciclos y diccionarios
- Contiene un menú interactivo

> [!NOTE]
> No tiene permanencia 


## Funcionalidades 
- El aplicativo permite registrar un nuevo producto en el inventario solicitando: nombre, precio, cantidad disponible.

- El aplicativo permite desplegar todos los productos registrado en el inventario con su información.

- El aplicativo permite modificar la cantidad disponible de un producto existente (el usuario debe ingresar el nombre del product y la nueva cantidad). 

- El aplicativo permite eliminar un producto existente por su nombre.

- El aplicativo permite calcular el total de los productos en el inventario.
